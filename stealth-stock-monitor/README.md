# Stealth Stock Monitor - 隐秘看盘

一款轻量级、跨平台的股票监控悬浮窗工具，支持 macOS 和 Windows。

## ✨ 核心功能

### 已实现功能

#### 🪟 窗口管理
- ✅ **自适应窗口大小** - 根据显示的股票数量和字体大小自动调整窗口尺寸
- ✅ **拖拽移动** - 点击窗口任意位置即可拖动调整位置
- ✅ **置顶显示** - 始终显示在其他窗口之上
- ✅ **透明度调节** - 可在设置中调整窗口透明度（10%-100%）
- ✅ **鼠标穿透模式** - 开启后窗口不响应鼠标点击，仅作展示
- ✅ **隐藏任务栏图标** - 不占用任务栏空间，通过系统托盘控制

#### 📊 股票监控
- ✅ **多股票监控** - 支持添加多只股票到自选列表
- ✅ **实时行情** - 自动刷新股票价格、涨跌幅等数据
- ✅ **自定义别名** - 可为股票设置简短易记的别名
- ✅ **显示行数控制** - 设置悬浮窗同时显示的股票数量（1-10行）
- ✅ **自动轮播** - 当股票数量超过显示行数时，自动循环播放展示
- ✅ **数据源切换** - 支持新浪、腾讯、东方财富三个数据源，故障时自动切换

#### ⚙️ 个性化配置
- ✅ **字体大小** - 小/中/大三档可选
- ✅ **涨跌颜色** - 自定义上涨/下跌/平盘的显示颜色
- ✅ **主题模式** - 支持自动/深色/浅色主题
- ✅ **刷新间隔** - 可调整数据刷新频率（1-60秒）
- ✅ **开机自启** - 可设置开机自动启动
- ✅ **暂停刷新** - 隐藏窗口时可选择暂停数据刷新以节省资源

#### ⌨️ 快捷键支持
- ✅ **显示/隐藏窗口** - 默认 `Cmd/Ctrl + Shift + S`
- ✅ **下一只股票** - 默认 `Alt + J`（当显示行数 < 总股票数时）
- ✅ **上一只股票** - 默认 `Alt + K`（当显示行数 < 总股票数时）
- ✅ **自定义快捷键** - 所有快捷键均可在设置中重新录制

#### 🎯 交互方式
- ✅ **右键菜单** - 在悬浮窗上右键弹出原生系统菜单（刷新/设置/隐藏/退出）
- ✅ **系统托盘** - 左键点击托盘图标显示/隐藏窗口，右键打开托盘菜单
- ✅ **设置窗口** - 完整的图形化配置界面，支持所有功能的可视化配置

## 🚀 使用方法

### 开发模式运行
```bash
npm run tauri dev
```

### 构建生产版本
```bash
npm run tauri build
```

## 📝 配置说明

配置文件自动保存在系统目录：
- **macOS**: `~/Library/Application Support/com.wolf.stealth-stock-monitor/config.json`
- **Windows**: `%APPDATA%/com.wolf.stealth-stock-monitor/config.json`

所有配置通过设置窗口修改后会自动保存并实时生效。

## 🎨 技术栈

- **前端**: React + TypeScript + Vite
- **后端**: Rust + Tauri 2.x
- **数据源**: 新浪财经 / 腾讯证券 / 东方财富 API

## 📌 注意事项

1. **鼠标穿透模式** - 开启后窗口无法点击，需通过快捷键或系统托盘进入设置关闭
2. **快捷键冲突** - 如果快捷键与其他应用冲突，可在设置中重新录制
3. **数据源切换** - 当主数据源连续失败3次后会自动切换到备用数据源
4. **股票代码格式** - 输入6位数字代码即可，系统会自动识别沪深市场

## 🔧 故障排除

- **窗口不显示** - 检查是否被隐藏，使用快捷键或托盘图标唤出
- **数据不更新** - 检查网络连接，或手动刷新（右键菜单 → 刷新）
- **快捷键无效** - 检查是否与其他应用冲突，尝试重新设置快捷键
