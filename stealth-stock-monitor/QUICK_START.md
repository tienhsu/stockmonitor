# 快速配置指南

## 🎯 显示多只股票

当前您的悬浮窗只显示1只股票。要显示多只股票，有两种方法：

### 方法1：使用设置窗口（推荐）✨

1. **打开设置窗口**
   - 右键点击悬浮窗 → 选择"设置"
   - 或点击系统托盘图标 → 右键 → 设置

2. **添加股票**（在"自选股"标签页）
   - 在输入框中输入股票代码，例如：
     - `000001` - 平安银行
     - `600036` - 招商银行  
     - `000858` - 五粮液
   - 点击"添加"按钮
   - 可以添加任意多只股票

3. **设置显示行数**（在"外观"标签页）
   - 找到"显示行数"选项
   - 拖动滑块或输入数字（1-10）
   - 例如设置为 `3` 就会同时显示3只股票

4. **保存并查看效果**
   - 设置会自动保存
   - 悬浮窗会立即更新显示

### 方法2：直接修改配置文件

如果您想快速配置，可以直接替换配置文件：

```bash
# 1. 备份当前配置
cp ~/Library/Application\ Support/com.wolf.stealth-stock-monitor/config.json \
   ~/Library/Application\ Support/com.wolf.stealth-stock-monitor/config.json.backup

# 2. 使用新配置（已包含3只股票，显示3行）
cp /tmp/update_config.json \
   ~/Library/Application\ Support/com.wolf.stealth-stock-monitor/config.json

# 3. 重启应用
# 关闭应用后重新运行 npm run tauri dev
```

新配置包含：
- ✅ 3只股票：茅台、平安银行、招商银行
- ✅ 显示行数：3行
- ✅ 其他设置保持不变

## ⌨️ 快捷键说明

### 当前已配置的快捷键：

1. **显示/隐藏窗口**：`Cmd + Shift + S`（Mac）或 `Ctrl + Shift + S`（Windows）
   - 按一次隐藏，再按一次显示
   - ✅ 这个功能已经实现

2. **下一只股票**：`Alt + J`
   - 当股票数量 > 显示行数时，向下滚动查看下一只
   - 例如：有5只股票，显示3行，按此键可以滚动查看后面的股票

3. **上一只股票**：`Alt + K`
   - 向上滚动查看上一只股票

### 修改快捷键：

在设置窗口的"快捷键"标签页：
1. 点击要修改的快捷键输入框
2. 按下新的组合键
3. 自动保存
