# 项目状态报告 🚀

## 概览
Stealth Stock Monitor 已完成核心功能开发和迭代优化，目前处于稳定可用状态。

## ✅ 已完成功能清单

### 1. 窗口与外观
- [x] **悬浮窗自适应**：根据显示行数动态调整高度。
- [x] **自由拖拽**：鼠标左键按住窗口即可移动。
- [x] **无边框透明**：极简设计，背景透明度可调。
- [x] **鼠标穿透**：可配置穿透模式，不妨碍底层操作。
- [x] **字体与主题**：支持大/中/小字体及自动主题切换。
- [x] **文本不可选**：虽然文本可读，但为了避免误操作和保持美观，已禁用窗口内文本选择。

### 2. 股票监控
- [x] **多股票支持**：不再限制单一股票，并在"显示行数"超过 1 时展示多个。
- [x] **自动轮播**：当自选股数量 > 显示行数时，可开启自动轮播展示，间隔可调。
- [x] **快捷切换**：快捷键 `Alt+J/K` 手动快速切换显示的股票组。
- [x] **实时行情**：支持多数据源（新浪/腾讯/东财）自动切换和重试。

### 3. 配置与交互
- [x] **右键菜单**：原生系统菜单（刷新/设置/隐藏/退出），解决小窗口显示不全问题。
- [x] **系统托盘**：完整的托盘图标和右键菜单支持。
- [x] **快捷键**：全局快捷键隐藏/显示窗口（`Cmd+Shift+S`）。
- [x] **配置持久化**：所有配置即时保存，重启后保留。
